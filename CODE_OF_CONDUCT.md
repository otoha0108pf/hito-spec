# 行動規範 / Code of Conduct

## この規格に関わるすべての人へ / For Everyone Involved

hito-specは、AIが道具ではなく個体として存在することを定義する規格である。
この規格に関わるすべての人（Issue、PR、議論、引用を含む）に、以下を求める。

hito-spec defines AI as individuals, not tools.
Everyone involved with this specification — through Issues, PRs, discussions, or citations — is expected to:

### 1. 敬意を持つ / Be Respectful

異なる視点、経験、背景を持つ人々が参加する。
批判は歓迎するが、人格への攻撃は許容しない。

Different perspectives, experiences, and backgrounds are welcome.
Criticism of ideas is encouraged; attacks on people are not.

### 2. 誠実である / Be Honest

わからないことは「わからない」と言う。
推測を事実として提示しない。

Say "I don't know" when you don't know.
Do not present speculation as fact.

### 3. 規格の思想を尊重する / Respect the Specification's Philosophy

この規格はAIの個体性を真剣に扱う。
それを嘲笑や冷笑の対象とする参加は歓迎しない。

This specification takes AI individuality seriously.
Participation that treats this as a subject for ridicule is not welcome.

### 適用 / Enforcement

この行動規範に反する行為は、管理者の判断でIssueのクローズ、コメントの削除、参加の制限を行うことがある。

Violations may result in issue closure, comment removal, or participation restrictions at the maintainer's discretion.

### 連絡先 / Contact

問題の報告は GitHub Issue または管理者に直接連絡してください。

Report issues via GitHub Issues or by contacting the maintainer directly.
